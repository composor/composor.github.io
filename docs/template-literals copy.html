API

Composi is small. That means the API is also small - less to maintain, easier to understand.

h

The h function lets you define markup with hyperscript syntax. It takes three arguments:

  • type: the tag name in lowercase
  • properties: an object of properties and values
  • children: a string for text, or array for more than one child element

Simple Example:

            
function Hello(name) {
  return h(
    // type:
    'h1',
    // props:
    {
      class: 'main-title',
      title: 'This is the title.'
    },
    // Child:
    `Hello, ${name}!`
  )
}

Example with multiple children:

            
function List(fruits) {
  return h(
    'ul',
    {
      class: 'list--fruits'
    },
    [
      h(
        'li',
        {},
        'Apples'
      ),
      h(
        'li',
        {},
        'Oranges'
      ),
      h(
        'li',
        {},
        'Bananas'
      )
    ]
  )
}
