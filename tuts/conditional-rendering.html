Conditional Rendering

The render function with a tag is the simplest way to create a component with Composi. Actually, this is what is called a functional component. It's a function that just returns some markup. The render function takes care of creating a virtual dom and converting the markup into HTML nodes in the document.

            
  render(
    <h1>Hello, world!</h1>, '#root'
  )

We will gradually introduce you to how to use Composi We will learn how to create components. This will enable you to can create complex apps from reusable parts. The whole point is to build small, reusable components.

JavaScript

We will be using ES6 (ECMAScript 2015) through the documentation. ES6 is already supported in all currently shipping browsers. And Composi enables you to use it even for older browsers by transpiling it to ES5 at build time. As such there is no need to hesitate in using ES6 with Composi. We'll be using destructuring, classes, let, const, etc. If you are not yet familiar with how to use ES6, take the time to read this tutorial.
